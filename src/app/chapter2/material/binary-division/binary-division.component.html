การหารเลขฐานสอง (Binary Division)

            การหารใช้หลักการเดียวกับการหารเลขฐานสิบ แต่การหารเลขฐานสองใช้กฎ 2 ข้อดังนี้
        (David, Money Harris and Sarah, L. Harris, 2007 : 247)
                    1) 0 หาร 1 = 0
                    2) 1 หาร 1 = 1
        
            จากผลลัพธ์ของการหารเลขฐานสอง ทดลองเปลี่ยนเป็นเลขฐานสิบ การทดสอบจะต้องได้
        ค่าที่ใกล้เคียงกับ 2.416...
        ทดสอบ
        1.11101110 = 
                      
                   = 1+ 0.5 + 0.25 + 0.125 + 0 + 0.03125 + 0.015625 + 0.007812
                      + 0.001953125
                   = 2.416
